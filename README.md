# Shop Framework
Functional implementation while abstracting design to a minima.

## Features
- Landing Page
- Shop
- Cart
- Checkout
- Register
- Login 
- Dashboard
...

## Stack
- SvelteKit (With TypeScript and SCSS.)
- NodeJS (Will migrate to Bun in the future.)
- Supabase (Will migrate to Turso in the future.)
- Figma (UI/UX design.)
